1. https://fakeimg.pl

https://fakeimg.pl/{分辨率}/{背景色}/{字体色}/?text={文字内容}&{可选项}

可选字体有font=lobster、font=bebas、font=museo；
使用font=noto以支持中文、日文或韩文。

使用font_size改变字体大小，如font_size=20。

使用retina=1会使图片放大一倍。

可选透明度(0-255)，如https://fakeimg.pl/350x200/ff0000,128/000,10。
